Site-wide wp-admin dashboard widget with a CoverKit generated image as the background.

== Description ==

Registers a **Dashboard widget** use case in the CoverKit template editor. Enable it on one template site-wide, map site fields (title, tagline, logo), and every admin sees a branded dashboard widget with the generated image as a CSS background.

Requires the main CoverKit plugin.
